How to use:

Scan

  • Use a flatbed scanner to scan the document.
  • Ensure you select the legal-size setting.
  • Save the scan as a Photo/JPEG file format.

Upload

  • Click the "Choose file" button to upload the JPEG file you scanned.

Edit

  • Once uploaded, input number of textboxes to generate then click "Generate" button.
  • Click and drag the textbox to the appropriate fields that need to be filled or corrected (e.g., loan amount, borrower details, etc.).
  • Note: Select then right click the textbox to delete. You can also double click the textbox to change the font size.

Print (Ctrl + P)

  • In the browser's print dialog, consider setting the "Margin" to None.
  • Set the printer to legal size to ensure the formatting remains correct.

Textbox shortcuts

  • Change size font size (Double click)
  • Align Left (Alt + L)
  • Center (Alt + E)
  • Align Right (Alt + R)
  • Justify (Alt + J)
